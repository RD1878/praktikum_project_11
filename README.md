## Инструкция по развертыванию проекта
1. Склонировать репозиторий в локальную директорию
`git clone https://github.com/RD1878/praktikum_project_11.git`
2. Установить NPM
`npm install`
3. Запустить с помощью локального сервера
`npm run dev`
